## 문제

[문제8 : 객체의 키 이름 중복](https://www.notion.so/8-c59d74ba23ed46b68d304fb03ac1b341) 

자바스크립트 객체를 다음과 같이 만들었다. 

출력값을 입력하시오. (출력값은 공백을 넣지 않습니다. )

```jsx
var d = {
    'height':180,
    'weight':78,
    'weight':84,
    'temperature':36,
    'eyesight':1
};

console.log(d['weight']);
```

## 제출 답안

```jsx
// ? 84
```

객체의 키가 weight인 값이 두개, 아래에 선언된 걸 가져올 것

## 공식 답안

정답은 '84' 입니다.
