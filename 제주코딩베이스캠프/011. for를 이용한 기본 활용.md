## 문제

[문제11 : for를 이용한 기본 활용](https://www.notion.so/11-for-e1aa3b5776fb4aa5b04addd81514f3a4)

1부터 100까지 모두 더하는 Code를 <pass> 부분에 완성하세요. `for`를 사용해야 합니다.

```jsx
let s = 0;

//pass

console.log(s);
```

## 제출 답안

```jsx
for(let i=1; i<=100; i++){
	s += i;
}
```

## 공식 답안

```jsx
let s = 0;

for (let i=1; i<=100; i++){
  s += i;
}

console.log(s);
```
