## 문제

[문제21 : set은 어떻게 만드나요?](https://www.notion.so/21-set-c8ff3a241cca477ab0c56d7e9a49e6b4)

다음 중 set을 만드는 방법으로 올바른 것을 모두 고르시오.

1.  var x = {1, 2, 3, 5, 6, 7};

2.  var x = {};

3.  var x = new Set('javascript');

4.  var x = new Set(range(5));

5.  var x = new Set();

## 제출 답안

3, 4, 5

## 공식 답안

3, 5

3.  var x = new Set('javascript');

4.  var x = new Set();
