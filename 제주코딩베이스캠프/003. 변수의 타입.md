## 문제

[문제3 : 변수의 타입](https://www.notion.so/3-8c063316324b42239786fd2c3e7bc105) 

다음 출력 값으로 올바른 것은?

```jsx
var arr = [100, 200, 300];
console.log(typeof(arr));
```

1)  undefined

2)  string

3)  number

4)  object

## 제출 답안

4) object

array는 object의 일부

## 공식 답안

4)  object

정답은 '4번'입니다. 

undefined, string, number 는 모두 기본 자료형(primitive type) 입니다.
